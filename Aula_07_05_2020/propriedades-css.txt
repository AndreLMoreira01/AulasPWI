-> Para texto:
color: define o valor da cor
font-size: define o tamanho da fonte
font-weight: é o peso da fonte
font-family: tipo de fonte ou seja sua família
text-align: alinha o texto para a esquerda
text-decoration: é usada para definir a formatação de  underline, overline, line-through ou blink
line-height: controlar o tamanho entre as linhas de um texto
font-style: permite fazer uma lista de propriedades das familias das fontes a serem especificados em um elemento selecionado

-> layout:
background: é um atalho para definir os valores de fundo individuais em um único lugar
margin: define a área de margem nos quatro lados do elemento
padding: define uma a distância entre o conteúdo de um elemento e suas bordas
position: ela serve para inserir a posição do elemento em uma determina coordenada(Position Fixes), o position relative posiciona o elemento em relação a si mesmo, o position abslute envolve outro position que tenha uma coordenada definida 
display: ela controla o layout de cada elemento
top: especifica o posicionamento vertical do elemento de acordo com o topo
bottom: define a margin area na parte inferior de um elemento
left: é um atalho que define os valores de border-left-color , border-left-style , e border-left-width
right: é um atalho que define os valores para o lado direito do elemento
height: determina a altura da área do conteúdo de um elemento
width: determina a largura da área de conteúdo de um elemento
float: determina que um elemento deve ser retirado do seu fluxo normal e colocado ao longo do lado direito ou esquerdo
border:  é uma propriedade para ajustar os valores de propriedades individuais da borda em um único elemento na folha do estilo

-> listas:
list-style: é uma propriedade usada para configurar list-style-type que no caso é uma propriedade para listas

-> unidades
px -> pixels
pt -> pontos
% -> porcentagem
em -> tamanho padrão
