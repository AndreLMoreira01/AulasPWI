Tags Html

!Doctype o !DOCTYPE, é utilizado para para indicar a versão do HTML que esta sendo utilizada
Html A tag Html(Html /Html) é simplesmente a estrutura básica, ela vai conter tood o seu texto e os elemntos utilizados para o formar.
Head A tag Head(Head /Head) é o cabeçalho do texto, não transmite nenhuma informação visível.
Title O title(title /title) é a tag que dá o nome a paigina ou guia
/meta A meta(/meta), é a tag que permite o uso de acentos utilizando o código charset=”UTF-8″.
body O body(body /body), essa teg representa o corpo do documento. Ela é responsavél por todos os elementos do texto.

Atributos

Os IDs, são utilizados para idicar isso ou aquilo ou seja algo específico, não podem ser reutizados por isso são unicos.

O style contém, informações de estilo para um documento ou uma parte de um documento.

O class é uma lista das classes de um elemento, separada por espaços.
